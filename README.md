# РГР №3. Регрессионные модели и их интерпретация

## Вариант C-6

**Тема:** построение и сравнение регрессионных моделей зависимости результативного признака от одного фактора.

По заданию РГР необходимо построить диаграмму рассеяния, сформулировать гипотезу о характере связи, построить линейную, квадратичную и дополнительную модель, сравнить их по качеству, выполнить подробный статистический анализ линейной модели и сделать прогноз для заданного значения фактора. В методических материалах указано, что для каждой модели нужно найти оценки параметров, записать итоговое уравнение, построить график модели и получить прогнозные значения. ## 3. Диаграмма рассеяния и гипотеза

![Диаграмма рассеяния](scatter.png)

По диаграмме рассеяния можно предположить, что между числом ядер процессора и TDP существует положительная зависимость. При увеличении числа ядер теплопакет процессора в среднем возрастает.

При этом точки расположены не строго вдоль одной прямой. Для одинакового числа ядер встречаются разные значения TDP, потому что теплопакет зависит не только от числа ядер, но и от архитектуры, поколения процессора, частоты, класса устройства и других характеристик.

Поэтому для описания зависимости целесообразно сравнить несколько моделей: линейную, квадратичную и степенную.

## 4. Линейная модель

Линейная модель имеет вид:

\[
\hat y = a + bx
\]

где:

- \(x\) — число ядер процессора;
- \(\hat y\) — прогнозируемое значение TDP;
- \(a\) — свободный коэффициент;
- \(b\) — коэффициент при факторе \(x\).

Для оценки коэффициентов линейной регрессии используем метод наименьших квадратов. В теоретической справке РГР для линейной модели указаны формулы: \(\hat b = \frac{S_{xy}}{S_{xx}}\), \(\hat a = \overline{y} - \hat b\overline{x}\).  [oai_citation:1‡Содержание РГР 3.pdf](sediment://file_0000000009b071fdb1731afeda50e893)

Формулы:

\[
\overline{x} = \frac{1}{n}\sum_{i=1}^{n}x_i
\]

\[
\overline{y} = \frac{1}{n}\sum_{i=1}^{n}y_i
\]

\[
S_{xx} = \sum_{i=1}^{n}(x_i - \overline{x})^2
\]

\[
S_{xy} = \sum_{i=1}^{n}(x_i - \overline{x})(y_i - \overline{y})
\]

\[
\hat b = \frac{S_{xy}}{S_{xx}}
\]

\[
\hat a = \overline{y} - \hat b\overline{x}
\]

По данным:

\[
\overline{x} = 8.8000
\]

\[
\overline{y} = 88.7667
\]

\[
S_{xx} = 7483.6000
\]

\[
S_{xy} = 35533.2000
\]

Находим коэффициент \(b\):

\[
\hat b = \frac{35533.2000}{7483.6000}
\]

\[
\hat b = 4.7481
\]

Находим коэффициент \(a\):

\[
\hat a = 88.7667 - 4.7481 \cdot 8.8000
\]

\[
\hat a = 46.9830
\]

Итоговое уравнение линейной регрессии:

\[
\hat y = 46.9830 + 4.7481x
\]

Коэффициент \(b\) положительный. Это означает, что при увеличении числа ядер на 1 TDP в среднем увеличивается примерно на 4.7481 Вт.

Коэффициент \(a\) формально показывает значение модели при \(x = 0\), но процессора с нулём ядер не бывает, поэтому содержательно этот коэффициент отдельно не интерпретируется.

### Качество линейной модели

Остаток:

\[
e_i = y_i - \hat y_i
\]

Остаточная сумма квадратов:

\[
RSS = \sum_{i=1}^{n} e_i^2
\]

Общая сумма квадратов:

\[
TSS = \sum_{i=1}^{n}(y_i - \overline{y})^2
\]

Коэффициент детерминации:

\[
R^2 = 1 - \frac{RSS}{TSS}
\]

Средняя ошибка аппроксимации:

\[
A = \frac{100\%}{n}\sum_{i=1}^{n}\left|\frac{y_i-\hat y_i}{y_i}\right|
\]

Для линейной модели:

\[
RSS = 69634.0325
\]

\[
TSS = 238350.7333
\]

\[
R^2 = 0.7079
\]

\[
A = 65.5857\%
\]

Коэффициент детерминации показывает, какая доля изменчивости зависимой переменной объясняется моделью. В приложении РГР указано, что \(R^2 = 1 - \frac{RSS}{TSS}\).  [oai_citation:2‡Содержание РГР 3.pdf](sediment://file_0000000009b071fdb1731afeda50e893)

Следовательно, линейная модель объясняет примерно 70.79% изменчивости TDP.

## 5. Квадратичная модель

Квадратичная модель имеет вид:

\[
\hat y = a + bx + cx^2
\]

Эта модель позволяет учитывать возможную нелинейность связи между числом ядер и TDP.

Для нахождения коэффициентов \(a\), \(b\), \(c\) используется метод наименьших квадратов. Система нормальных уравнений имеет вид:

\[
\begin{cases}
na + b\sum x_i + c\sum x_i^2 = \sum y_i, \\
a\sum x_i + b\sum x_i^2 + c\sum x_i^3 = \sum x_iy_i, \\
a\sum x_i^2 + b\sum x_i^3 + c\sum x_i^4 = \sum x_i^2y_i.
\end{cases}
\]

По данным были рассчитаны суммы:

\[
n = 60
\]

\[
\sum x_i = 528
\]

\[
\sum x_i^2 = 12130
\]

\[
\sum x_i^3 = 494076
\]

\[
\sum x_i^4 = 25217650
\]

\[
\sum y_i = 5326
\]

\[
\sum x_iy_i = 82402
\]

\[
\sum x_i^2y_i = 2730540
\]

Подставляем значения в систему:

\[
\begin{cases}
60a + 528b + 12130c = 5326, \\
528a + 12130b + 494076c = 82402, \\
12130a + 494076b + 25217650c = 2730540.
\end{cases}
\]

Система была решена методом Гаусса.

После исключения коэффициента \(a\) из второго и третьего уравнений получаем:

\[
7483.6b + 387332c = 35533.2
\]

\[
387332b + 22765368.3333c = 1653800.3333
\]

Затем исключается коэффициент \(b\), после чего получается уравнение:

\[
2718054.44c = -185307.238
\]

Отсюда:

\[
c = -0.0682
\]

Обратной подстановкой получаем:

\[
b = 8.2768
\]

\[
a = 29.7140
\]

Итоговое уравнение квадратичной модели:

\[
\hat y = 29.7140 + 8.2768x - 0.0682x^2
\]

Коэффициент при \(x\) положительный, поэтому при увеличении числа ядер TDP в среднем возрастает. Коэффициент при \(x^2\) отрицательный, поэтому парабола направлена вниз. Это означает, что рост TDP при увеличении числа ядер описывается не строго прямой линией.

### Качество квадратичной модели

Для квадратичной модели:

\[
RSS = 57000.4473
\]

\[
R^2 = 0.7609
\]

\[
A = 58.1309\%
\]

Квадратичная модель объясняет примерно 76.09% изменчивости TDP.

## 6. Степенная модель

Дополнительная модель по варианту C-6 — степенная:

\[
\hat y = ax^b
\]

Эта модель нелинейная, поэтому для оценки параметров используется линеаризация. В приложении РГР указано, что степенная модель приводится к линейному виду с помощью преобразования:

\[
\ln y = \ln a + b\ln x
\]

 [oai_citation:3‡Содержание РГР 3.pdf](sediment://file_0000000009b071fdb1731afeda50e893)

Вводим новые переменные:

\[
X = \ln x
\]

\[
Y = \ln y
\]

\[
A = \ln a
\]

Тогда модель принимает линейный вид:

\[
Y = A + bX
\]

Логарифмирование допустимо, так как во всех наблюдениях:

\[
x > 0
\]

\[
y > 0
\]

По данным:

\[
\overline{X} = 1.7241
\]

\[
\overline{Y} = 4.2216
\]

\[
S_{XX} = 45.1474
\]

\[
S_{XY} = 29.3544
\]

Коэффициент \(b\):

\[
b = \frac{S_{XY}}{S_{XX}}
\]

\[
b = 0.6502
\]

Коэффициент \(A\):

\[
A = \overline{Y} - b\overline{X}
\]

\[
A = 3.1006
\]

Так как:

\[
A = \ln a
\]

то:

\[
a = e^A
\]

\[
a = e^{3.1006}
\]

\[
a = 22.2110
\]

Итоговая степенная модель:

\[
\hat y = 22.2110x^{0.6502}
\]

Так как показатель степени положительный, модель показывает положительную зависимость между числом ядер и TDP. При этом \(0 < b < 1\), поэтому рост TDP происходит с замедляющимся темпом.

### Качество степенной модели

Для степенной модели:

\[
RSS = 62868.6927
\]

\[
R^2 = 0.7362
\]

\[
A = 50.6625\%
\]

По совокупности критериев основной моделью выбирается квадратичная модель, потому что она имеет наибольшее \(R^2\), наименьшее \(RSS\) и хорошо описывает нелинейный характер зависимости.

## 9. Анализ остатков

Остаток рассчитывается по формуле:

\[
e_i = y_i - \hat y_i
\]

Если остатки расположены около нуля без ярко выраженной закономерности, модель описывает данные лучше.

### Остатки линейной модели

![Остатки линейной модели](residuals_linear.png)

Для линейной модели остатки имеют заметный разброс. Есть как положительные, так и отрицательные остатки. Это означает, что линейная модель не всегда хорошо описывает значения TDP, особенно для отдельных процессоров с большим числом ядер.

### Остатки квадратичной модели

![Остатки квадратичной модели](residuals_quad.png)

У квадратичной модели остатки в целом имеют меньшую сумму квадратов, чем у линейной модели. Это подтверждается значением \(RSS\). Поэтому квадратичная модель лучше приближает исходные данные.

### Остатки степенной модели

![Остатки степенной модели](residuals_power.png)

Степенная модель также описывает положительную связь между числом ядер и TDP. По средней относительной ошибке она показывает лучший результат, но по \(R^2\) и \(RSS\) уступает квадратичной модели.

## 10. Подробный статистический анализ линейной модели

По заданию подробный статистический анализ выполняется именно для линейной модели: нужно найти остатки, RSS, оценку дисперсии ошибки, доверительные интервалы для коэффициентов и проверить гипотезу о значимости коэффициента при \(x\).  [oai_citation:5‡Содержание РГР 3.pdf](sediment://file_0000000009b071fdb1731afeda50e893)

Линейная модель:

\[
\hat y = 46.9830 + 4.7481x
\]

Остаточная сумма квадратов:

\[
RSS = 69634.0325
\]

Несмещённая оценка дисперсии ошибки:

\[
s^2 = \frac{RSS}{n-2}
\]

\[
s^2 = \frac{69634.0325}{60-2}
\]

\[
s^2 = 1200.5868
\]

Стандартное отклонение ошибки:

\[
s = \sqrt{s^2}
\]

\[
s = 34.6495
\]

### Стандартные ошибки коэффициентов

Стандартная ошибка коэффициента \(b\):

\[
SE(\hat b) = \frac{s}{\sqrt{S_{xx}}}
\]

\[
SE(\hat b) = 0.4005
\]

Стандартная ошибка коэффициента \(a\):

\[
SE(\hat a) = s\sqrt{\frac{1}{n}+\frac{\overline{x}^2}{S_{xx}}}
\]

\[
SE(\hat a) = 5.6950
\]

Для 95% доверительных интервалов используется квантиль распределения Стьюдента:

\[
t_{crit} = 2.0017
\]

Число степеней свободы:

\[
df = n - 2 = 58
\]

В методических материалах доверительные интервалы для коэффициентов линейной модели задаются формулами:

\[
\hat \theta_1 \pm tSE(\hat \theta_1)
\]

\[
\hat \theta_0 \pm tSE(\hat \theta_0)
\]

 [oai_citation:6‡Содержание РГР 3.pdf](sediment://file_0000000009b071fdb1731afeda50e893)

### Доверительный интервал для \(a\)

\[
\hat a = 46.9830
\]

\[
SE(\hat a) = 5.6950
\]

\[
\hat a \pm t_{crit}SE(\hat a)
\]

\[
46.9830 \pm 2.0017 \cdot 5.6950
\]

\[
a \in [35.5832; 58.3829]
\]

\[
\hat b = 4.7481
\]

\[
SE(\hat b) = 0.4005
\]

\[
\hat b \pm t_{crit}SE(\hat b)
\]

\[
4.7481 \pm 2.0017 \cdot 0.4005
\]

\[
b \in [3.9464; 5.5499]
\]

Так как доверительный интервал для \(b\) не содержит 0, коэффициент при \(x\) является статистически значимым на уровне значимости \(\alpha = 0.05\).

## 11. Проверка гипотезы о значимости коэффициента \(b\)

Проверяем гипотезы:

\[
H_0: b = 0
\]

\[
H_1: b \ne 0
\]

Если \(H_0\) верна, то число ядер не оказывает статистически значимого влияния на TDP в рамках линейной модели.

Наблюдаемая статистика:

\[
t_{\text{набл}} = \frac{\hat b}{SE(\hat b)}
\]

\[
t_{\text{набл}} = \frac{4.7481}{0.4005}
\]

\[
t_{\text{набл}} = 11.8545
\]

Критическое значение:

\[
t_{\text{крит}} = 2.0017
\]

Так как:

\[
|t_{\text{набл}}| > t_{\text{крит}}
\]

\[
11.8545 > 2.0017
\]

то гипотеза \(H_0\) отвергается.

Также:

\[
p\text{-value} < 0.0001
\]

Следовательно, коэффициент \(b\) статистически значим. Это означает, что в линейной модели число ядер статистически значимо связано с TDP процессора.

Округлённо:

\[
\hat y(20) \approx 168 \text{ Вт}
\]

То есть для процессора AMD с 20 ядрами ожидаемый TDP по выбранной модели составляет примерно 168 Вт.

## 13. Итоговый вывод

В работе исследовалась зависимость TDP процессоров AMD от числа ядер на основе среза набора данных `openintro::cpu`. Фактором являлось число ядер \(x\), результативным признаком — TDP \(y\) в ваттах.

По диаграмме рассеяния была выявлена положительная зависимость: при росте числа ядер TDP в среднем увеличивается. Были построены три модели: линейная, квадратичная и степенная.

Линейная модель имеет вид:

\[
\hat y = 46.9830 + 4.7481x
\]

Квадратичная модель имеет вид:

\[
\hat y = 29.7140 + 8.2768x - 0.0682x^2
\]

Степенная модель имеет вид:

\[
\hat y = 22.2110x^{0.6502}
\]

Наилучшее качество по \(R^2\) и \(RSS\) показала квадратичная модель. Для неё:

\[
R^2 = 0.7609
\]

\[
RSS = 57000.4473
\]

Линейная модель также подтвердила статистическую значимость связи, поскольку доверительный интервал для коэффициента \(b\) не содержит ноль, а гипотеза \(H_0: b = 0\) была отвергнута.

Для \(x^* = 20\) основной прогноз по выбранной квадратичной модели равен:

\[
\hat y(20) = 167.9790
\]

Следовательно, для процессора AMD с 20 ядрами ожидаемый TDP составляет примерно 168 Вт.
